1. Introduction to Python

Python is a high-level, interpreted programming language known for its readability and simplicity. It's widely used in web development, data science, artificial intelligence, and more.

Two Ways to Use Python:
  • Interactive Shell: Type commands one at a time (great for learning)
  • Script Files: Write complete programs in .py files

First Python Code:

>>> print("Hello, World!") Hello, World! >>> 2 + 2 4

2. Expressions, Values, and Operators

Expression: A combination of values and operators that evaluates to a single value.

Value: Basic data like numbers or text.

Operator: Symbols that perform operations (+, -, *, etc.).

Python Math Operators:

Operator Operation Example Result
** Exponent 2 ** 3 8
% Modulus/Remainder 22 % 8 6
// Integer Division 22 // 8 2
/ Division 22 / 8 2.75
* Multiplication 3 * 5 15
- Subtraction 5 - 2 3
+ Addition 2 + 2 4

Operator Precedence (Order of Operations):

  1. Parentheses ()
  2. Exponents **
  3. Multiplication *, Division /, Integer Division //, Modulus % (left to right)
  4. Addition +, Subtraction - (left to right)
Examples: >>> 2 + 3 * 6 20 >>> (2 + 3) * 6 30 >>> 2 ** 3 + 4 12

3. Data Types

Every value in Python has a data type. The three basic types are:

A. Integers (int)

Whole numbers, positive or negative, without decimals.

# Examples of integers 42 -7 0 1000

B. Floating-Point Numbers (float)

Numbers with decimal points.

# Examples of floats 3.14 -0.5 2.0 0.0

C. Strings (str)

Text data enclosed in quotes.

# Examples of strings 'Hello' "Python" '123' # This is a string, not a number! '' # Empty string
Important: 42 (int) ≠ '42' (string) ≠ 42.0 (float)

String Operations:

# String Concatenation (Joining) >>> 'Hello' + ' ' + 'World' 'Hello World' # String Replication (Repeating) >>> 'Ha' * 3 'HaHaHa' # ERROR: Can't mix types >>> 'Number: ' + 42 TypeError: can only concatenate str (not "int") to str

4. Variables

Variables are named containers that store values in memory.

Assignment Statement:

# Creating variables age = 25 name = 'Alice' price = 19.99

Variable Naming Rules:

  1. Must be one word (no spaces)
  2. Can contain letters, numbers, and underscores
  3. Cannot start with a number
  4. Case-sensitive (myVar ≠ myvar)
Valid Names Invalid Names Why Invalid
age my age Space not allowed
age2 2age Can't start with number
user_name user-name Hyphen not allowed
_private $amount Special character

Using Variables:

>>> x = 10 >>> y = 5 >>> total = x + y >>> total 15 >>> x = 20 # Overwrite previous value >>> x 20

5. Basic Functions

The print() Function:

Displays output on the screen.

print('Hello, World!') print(42) print() # Prints empty line

The input() Function:

Gets user input. Always returns a string.

name = input('Enter your name: ') print('Hello, ' + name)

The len() Function:

Returns the length of a string (number of characters).

>>> len('hello') 5 >>> len('Python Programming') 18 >>> len('') 0

Type Conversion Functions:

Problem: Python doesn't automatically convert between types.
# ERROR: Can't concatenate string and integer 'I am ' + 29 + ' years old.'

Solution: Use conversion functions:

  • str() - Convert to string
  • int() - Convert to integer (rounds down floats)
  • float() - Convert to float
>>> str(42) # '42' >>> int('100') # 100 >>> int(7.8) # 7 (rounds down) >>> float('3.14') # 3.14 >>> float(10) # 10.0

Complete Example Program:

# Simple Python Program print('What is your name?') name = input() print('What is your age?') age = input() # age is a string! print('Hello, ' + name + '!') print('Next year you will be ' + str(int(age) + 1) + ' years old.')

6. Practice Exercises

Exercise 1: What will these evaluate to?
  1. (4 + 6) * 2
  2. 'Py' + 'thon'
  3. 'Na' * 4
  4. int(8.9)
  5. str(50) + ' points'
Answers:
  1. 20
  2. 'Python'
  3. 'NaNaNaNa'
  4. 8
  5. '50 points'
Exercise 2: Fix these errors
  1. print('Score: ' + 95)
  2. city = input()
print('City:' city)
  3. value = '75'
result = value * 2
Answers:
  1. print('Score: ' + str(95))
  2. print('City: ' + city) or print('City:', city)
  3. result = int(value) * 2 or result = value * 2 (if you want string repetition)
Exercise 3: Write a program that:
  • Asks for two numbers
  • Adds them together
  • Multiplies them
  • Displays both results
Solution: 
print('Enter first number:')
num1 = input()

print('Enter second number:')
num2 = input()

# Convert to integers for math operations
sum_result = int(num1) + int(num2)
product_result = int(num1) * int(num2)

print('Sum:', sum_result)
print('Product:', product_result)

Key Points

Important Rules to Remember:

  1. Python follows operator precedence (PEMDAS-like rules)
  2. Strings must be in quotes
  3. input() always returns a string
  4. You can't mix types in operations without conversion
  5. Variable names have specific rules
